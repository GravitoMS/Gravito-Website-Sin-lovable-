## 🔧 DESARROLLO RÁPIDO

```bash
# Instalar dependencias
npm install

# Desarrollo local
npm run dev

# Build para producción
npm run build

# Análisis de simplicidad
node scripts/complexity-analyzer.cjs

# Verificación de compatibilidad Lovable
node scripts/lovable-compatibility-check.cjs
```
